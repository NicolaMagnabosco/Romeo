Manage Datasets

Romeo allows the user to visualize the details of the stored Datasets and permits their removal. These features are assigned to the Datasets managing window. From the main window of Romeo, select the Manage Datasets button or choose the right one in the toolbar. The Datasets managing window (Fig. 14) is composed of two panels. The left one contains the list of Datasets actually stored in Romeo, while the right one contains the details of the group of Subjects and Protocols associated with the selected Dataset . In any moment, the user can go back to the main window by selecting the Back button.

Visualize Dataset details:

to visualize the details of a Dataset , follow these instructions:

  • From the Datasets managing window, select the Dataset of interest from the list;
  • The details panel will show informations about the groups of Subjects and the Protocols associated with the Dataset .

Delete a Dataset:

to delete a Dataset from Romeo, follow these instructions:

  • From the Datasets managing window, select the Dataset of interest from the list;
  • Select the Delete button to confirm the removal.
