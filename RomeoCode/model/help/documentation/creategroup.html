Create Group of Subject

The creation of a new group of Subjects is assigned to the group of Subjects creation window. From the main window of Romeo, select the New Group button or choose the right one in the toolbar. The group of Subjects creation window (Fig.6) is composed of two panels. The left one contains the information about the group that is going to be created (Name and Type). The right panel contains the list of Subjects actually stored in Romeo (Filtered according to the selected type), plus the utility buttons Select All and Deselect All. In any moment, the user can go back to the main window by selecting the Back button. To create a new group of Subjects , follow these instructions:

  • From the group of Subjects creation window, type a unique name in the Group name field. That name will be the name of the group;
  • Select the type of the Subjects that will be included in the group from the Type drop-down menu. A list of Subjects with that type, will be listed in the right panel;
  • Select the Subjects that have to be included into the group, from the list of Subjects ;
  • Confirm the operation by selecting the Save button. Romeo will report an error message if the name chosen for the group is already present in Romeo or not all the required fields are fulfilled.
