Manage Group of Subject

Romeo allows the user to visualize the details of the stored groups of Subjects and permits their removal or modification. These features are assigned to the groups of Subjects managing window. From the main window of Romeo, select the Manage Groups button or choose the right one in the toolbar. The groups of Subjects managing window (Fig. 7) , is composed of two panels. The left one contains the list of the groups of Subjects stored in Romeo, ordered by creation data (by default, from the oldest one, to the newest). For each group it is specified:

  • Name: the name of the group;
  • Type: the type of the Subjects associated with it (2D, 3D, 2D-t, 3D-t);
  • Subjects: the number of Subjects associated with it;
  • Creation Date: the date and time in which the group of Subjects was created.

The right panel contains the details about the Subjects of the selected group. In particular, it shows a list with their names and the flags of presence of the mask. In any moment, the user can go back to the main window by selecting the Back button.

Visualize group of Subjects details:

to visualize the groups of Subjects present into Romeo, follow these instructions:

  • Select the group for which you want to view the details, from the list present in the left panel of the window;
  • The Details panel will show the details of the Subjects associated with the group.

Modify a group of Subject:

to modify a group, follow these instructions:

  • From the groups of Subjects managing window, select the group that you want to modify;
  • Select the Edit button. Romeo will load the Fig. 8 window;
  • From this window it is possible to add or remove some Subjects from the group. Select or deselect the Subjects you want to add or remove, from the list in the right panel;
  • Confirm the operation by selecting the Save button./li>

Delete a group of Subject:

to delete a group of Subjects from Romeo, follow these instructions:

  • From the groups of Subjects managing window, select the group of interest from the list;
  • Select the Delete button to confirm the removal.
