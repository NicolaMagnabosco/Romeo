Manage Protocols

Romeo allows the user to visualize the details of the stored Protocols and permits their removal. These features are assigned to the Protocols managing window. From the main window of Romeo, select the Manage Protocols button or choose the right one in the toolbar. The Protocols managing window (Fig. 11) is composed of two panels. The left one contains the list of Protocols actually stored in Romeo, while the right one contains the details of the selected Protocol . In any moment, the user can go back to the main window by selecting the Back button.

Visualize Protocol details:

o visualize the details of a Protocol , follow these instructions:

  • From the Protocols managing window, select the Protocol of interest from the list;
  • The details panel will show information about the features extractors and/or cluster algorithm associated with the Protocol .

Modify a Protocol:

to modify a Protocol , follow these instructions:

  • From the Protocols managing window, select the one that you want to modify;
  • Select the Edit button. Romeo will load the Fig. 12 window;
  • From this window it is possible to add or remove some features and/or change the cluster algorithm of the protocol. To perform these actions, follow the instructions.
  • Confirm the operation by selecting the Save button.

Delete a Protocol:

to delete a Protocol from Romeo, follow these instructions:

  • From the Protocols managing window, select the Protocol of interest from the list;
  • Select the Delete button to confirm the removal.
