Create a Dataset

The creation of a new Dataset is assigned to the Dataset creation window. From the main window of Romeo, select the New Dataset button or choose the right one in the toolbar. The Dataset creation window (Fig. 13), is composed of three panels and a form. The form allow the user to type the name of the Dataset that will be created. The left panel contains the list of groups of Subjects actually stored in Romeo, while the center one contains the list of Protocols (filtered according to the selected group). The right panel contains the details of the selected group of Subject and Protocol . In any moment, the user can go back to the main window by selecting the Back button. To create a new Dataset , follow these instructions:

  • From the Dataset creation window, type a unique name in the Dataset Name field. That name will be the name of the Dataset ;
  • Select the group of Subjects that have to be associated with the Dataset from the roup Of Subjects panel. Romeo will load the compatible Protocols in the Protocols panel;
  • Select the Protocols that have to be associated with the Dataset from the Protocols panel;
  • Select the Save button to confirm the operation. Romeo will report an error message if the name chosen for the Dataset is already present in Romeo.
